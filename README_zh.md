# vst24pas

vst2.4 for pascal

Pascal语言实现的VST 2.4

你可以用它创建一个vst插件, 但是由于本人水平有限, 可能存在bug

## 安装和使用

支持Free Pascal和Delphi

具体请看例子

PS:

- FPC 3.2.0 和 3.3.1 测试 OK

- Delphi 10.3.3 社区版测试 OK

### lazarus

打开vst24pas.lpk, 编译

新建工程-库, 在需要的包处添加vst24pas

### delphi

添加source目录到search path
